# Roadmap

This roadmap is intentionally practical. It focuses on making VIDEW easier to deploy, easier to customize, and more credible as a source-available product starter.

## Near-Term Priorities

- tighten deployment documentation for shared hosting, VPS, and reverse-proxy setups
- improve screenshot coverage and public-facing repository presentation
- keep polishing the admin workflows for moderation, creator review, and video management
- improve empty states, onboarding cues, and first-run defaults in the admin and creator studio
- continue production hardening around billing, storage, and webhook handling

## Mid-Term Priorities

- richer video analytics for creators and admins
- more complete channel discovery and creator browsing flows
- broader moderation tools and queue management
- stronger import/export and backup-friendly admin utilities
- better theming and branding controls without requiring code changes

## Long-Term Vision

- a stronger "launch-ready" experience for niche video products
- a more complete creator economy workflow around profiles, growth, and monetization
- improved media pipeline options beyond basic upload and embed flows
- deeper deployment tooling and packaging for self-hosted teams
- a more polished public release experience with screenshots, example setups, and expanded docs
